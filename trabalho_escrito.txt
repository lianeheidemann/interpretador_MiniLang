# PyLite Interpreter

────────────────────────────

INTERPRETADOR DE LINGUAGEM SIMPLIFICADA

Este projeto implementa um interpretador de uma linguagem própria simplificada, desenvolvido em Python. Ele é capaz de realizar análise léxica, parsing e execução de estruturas básicas como variáveis, expressões e condicionais.

O objetivo do projeto é demonstrar, de forma prática, conceitos fundamentais de construção de linguagens e interpretadores.

────────────────────────────

VISÃO GERAL

A linguagem implementada inclui:

• Análise léxica (Lexer baseado em expressões regulares)  
• Parsing (organização e interpretação da estrutura do código)  
• Execução de comandos (Executor)  
• Variáveis dinâmicas  
• Operações matemáticas  
• Operadores lógicos  
• Estruturas condicionais (if / else)  
• Bloco principal (main { })  

────────────────────────────

ARQUITETURA

Fluxo de execução do interpretador:

Lexer → Parser → Executor

Descrição:

• Lexer: converte o código-fonte em tokens  
• Parser: organiza os tokens em estruturas interpretáveis  
• Executor: executa as instruções e controla o fluxo do programa  

────────────────────────────

COMO O PROJETO FUNCIONA

🔤 LEXER

Responsável por transformar o código em tokens, identificando:

• Números  
• Identificadores  
• Palavras-chave: var, if, else, main  
• Operadores: + - * / == != >= <= > < =  
• Símbolos: { } ( ) ; :  

────────────────────────────

🧠 PARSER

Responsável por interpretar a sequência de tokens, estruturando:

• Blocos de código  
• Expressões  
• Condicionais  
• Hierarquia das instruções  

────────────────────────────

🧩 EXECUTOR

Após o parsing, o executor:

• Executa comandos sequencialmente  
• Gerencia variáveis em memória (self.vars)  
• Avalia expressões matemáticas e lógicas  
• Controla o fluxo de execução (if / else e blocos)  

────────────────────────────

EXEMPLO DE CÓDIGO

main {
    var x: int = 10;
    var y: int = 5;

    x = x + 3;

    if x > 10 {
        y = y * 2;
    } else {
        y = 0;
    }
}
